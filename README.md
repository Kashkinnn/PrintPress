<h1 align="center">🖨️ PrintPress</h1>

<p align="center">
  <strong>From Figma Concept to Full-Stack Reality</strong>
</p>

PrintPress is a web-based print management portal that originated as a UI/UX concept in Figma and has been fully realized into a dynamic application. It uses **HTML** and **CSS** for a responsive, modern frontend, powered heavily by **PHP** and **MySQL** on the backend to handle user sessions, data processing, and hardware tracking.

## ✨ Key Features

Based on the core system architecture, PrintPress includes:
* **Role-Based Access Control (RBAC):** Distinct, secure dashboards tailored for **Admins**, **Staff**, and **Students**.
* **Student Portal:** Allows students to create print jobs, select color modes (B&W/Colored), track job status (Pending, Printing, Completed), and view their print history.
* **Admin Dashboard:** A comprehensive command center featuring real-time statistics on revenue, pending jobs, total users, and active stations.
* **Hardware & Station Tracking:** Monitors the operational status of print stations across campus and the payment methods they accept (GCash, PayMaya, Cash).
* **Inventory Management:** Tracks print supplies and triggers low-stock warnings when items fall below their designated reorder points.
* **Secure Authentication:** Features password hashing and secure session management for user registration and logins.

## 🛠️ Technologies Used

* **Frontend:** HTML5, CSS3 (Custom UI inspired by Figma designs)
* **Backend:** PHP (Vanilla)
* **Database:** MySQL
* **Local Server Environment:** XAMPP

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
* Download and install [XAMPP](https://www.apachefriends.org/index.html).
* Ensure you have a web browser and a code editor (like VS Code) installed.

### Installation

1. **Clone the repository**
   Navigate to your XAMPP `htdocs` folder (usually `C:\xampp\htdocs` on Windows or `/Applications/XAMPP/htdocs` on Mac) and clone the repo
